# Solo Leveling System 

A gamified habit tracking and productivity system inspired by the "Solo Leveling" concept.

## Features
- Habit tracking with start & end dates
- Calendar-based daily progress
- Daily to-do lists
- Gamification with EXP, levels, and ranks (in progress)

## Tech Stack
- Backend: Django
- Frontend: HTML, CSS, JavaScript
- Database: SQLite (development)
